# Student Registration

Multi-section student registration form with live validation, password feedback, and a notebook-inspired layout. Vanilla HTML, CSS, and JavaScript.

## Features

| Feature | Description |
|---------|-------------|
| Multi-fieldset form | Personal, academic, and account sections |
| Client validation | Required fields, patterns, password rules |
| Stepper / progress | Lightweight section guidance |
| Branding assets | PUP and program logos |

## Quick start

```bash
python -m http.server 8000
```

## Structure

```text
student-registration/
  index.html
  script.js
  style.css
  assets/
```
